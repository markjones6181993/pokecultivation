# ===============================================================================
# CRAFTING RECIPES
# ===============================================================================

# Each recipe follows the format:
# [RECIPE_ID]
# Name = Display Name
# Description = Recipe description
# Result = ITEM_ID,quantity
# Ingredients = ITEM_ID1,qty1,ITEM_ID2,qty2,ITEM_ID3,qty3...
# Category = category_name (optional)
# UnlockCondition = condition_string (optional - if not specified, recipe is always available)
#
# ===============================================================================
# SUPPORTED UNLOCK CONDITIONS:
# ===============================================================================
#
# badges >= X                    - Player has X or more gym badges
# story_flag("FLAG_NAME")        - Specific story flag is set to true
# has_item("ITEM_SYMBOL")        - Player has the specified item in bag
# pokemon_seen >= X              - Player has seen X or more Pokemon species
# pokemon_caught >= X            - Player has caught X or more Pokemon species
# level >= X                     - Player's highest level Pokemon is X or higher
# money >= X                     - Player has X or more money
# 
# Multiple conditions can be combined with commas (AND logic):
# UnlockCondition = badges >= 5, story_flag("SURF_UNLOCKED"), pokemon_seen >= 75
#
#===============================================================================
